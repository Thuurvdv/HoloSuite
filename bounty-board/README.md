# Bounty Board

A HoloSuite-compatible Foundry VTT module for sci-fi bounty contracts.

Enable the module in Foundry, then open it from the scene controls or HoloSuite Core launcher.

The public API is available at:

```js
game.scifiSuite.bountyBoard
game.modules.get("bounty-board").api
```

GM-only test data can be installed from the console:

```js
game.scifiSuite.bountyBoard.seedTestData()
```
